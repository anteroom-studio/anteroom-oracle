Quick Setup

Clone repository

git clone https://github.com/Zawwarsami16/ZAI-Oracle
cd ZAI-Oracle


Create environment file

cp .env.example .env
nano .env


Add your key

GROQ_KEY=gsk_your_key_here


Run Oracle

python3 zai_oracle.py

Command Modes
Live Intelligence Mode
python3 zai_oracle.py
